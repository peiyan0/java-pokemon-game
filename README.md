## Pokémon Battle Game

### Overview
This project is a Pokémon Battle Game implemented in Java. Players can catch Pokémon, engage in battles, and earn medals. The game includes various Pokémon types and a scoring system to rank players based on their battle performance.

### Features
- **Catch Pokémon**: Players can catch different types of Pokémon.
- **Battle System**: Engage in battles with caught Pokémon.
- **Medal System**: Earn medals based on battle performance.
- **Score Ranking**: View the ranking of battle scores.

### Project Structure
The project is organized into the following packages and classes:

#### Packages
- `Game`: Contains classes related to game mechanics such as battles and scenarios.
- `Model`: Contains classes representing the data models such as Pokémon and medals.

#### Key Classes
- `MainGame`: The main entry point of the game.
- `Pokemon`: Represents a Pokémon with attributes like name, type, HP, attack name, and attack damage.
- `PokemonList`: Manages the list of available Pokémon.
- `Medal`: Manages the medals earned by players.
- `Battle`: Handles the battle mechanics between Pokémon.
- `Catch`: Manages the catching of Pokémon.
- `Scenario`: Defines different scenarios in the game.
- `Score`: Manages and displays the ranking of battle scores.

#### Usage
- **Start the Game**: Follow the on-screen instructions to start the game.
- **View All Pokémon**: Display a list of all available Pokémon.
- **See Battle Scores**: View the ranking of battle scores.
- **Exit**: Exit the game.
